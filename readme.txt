Allows you to add a notification at the top of every page, similar to the Squarespace announcement bar.

== Description ==

The plugin allows to to show a notification bar at the top of every page. The notice can include a heading, message and link. Notices can also be scheduled to appear at a specific time.

Developed by [James Robinson](https://jmr.codes).

== Installation ==

1. Unzip the plugin and copy the `notice` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
